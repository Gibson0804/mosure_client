# HCMS Client 手机管理端设计文档

## 项目概述

**HCMS Client** 是 HCMS11 内容管理系统的移动端管理应用，基于 uni-app 开发，支持多端部署（iOS、Android、H5、小程序）。

### 核心定位

- **轻量化管理**：随时随地查看项目状态和内容
- **AI 智能助手**：通过对话式交互快速查询和操作项目
- **快捷入口**：一键跳转项目前端页面

### 技术栈

- **框架**：uni-app (Vue 3)
- **UI 组件**：uni-ui
- **状态管理**：Pinia
- **网络请求**：uni.request 封装
- **后端接口**：HCMS11 `/client/*` 系列 API
