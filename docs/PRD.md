# HCMS Client 产品需求文档 (PRD)

## 1. 产品背景

### 1.1 产品定位

HCMS Client 是面向 HCMS11 系统管理员和运营人员的移动端管理工具，解决以下痛点：

- 外出时无法及时查看项目状态
- 需要快速查询内容数据但不方便打开电脑
- 希望通过自然语言快速获取系统信息

### 1.2 目标用户

| 角色 | 使用场景 | 核心需求 |
|------|----------|----------|
| 系统管理员 | 日常巡检、问题排查 | 快速查看项目状态、任务执行情况 |
| 内容运营 | 内容审核、数据查询 | 浏览内容列表、查看详情 |
| 项目经理 | 项目跟进 | 查看项目概览、快速访问项目前端 |

### 2.2 核心功能详情

#### 2.2.1 登录模块

**功能描述**：支持多服务器切换登录

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 服务器地址输入 | 支持输入自定义域名，自动补全协议 | P0 |
| 账号密码登录 | 使用 HCMS 系统账号登录 | P0 |
| 记住登录 | 本地保存 Token，支持自动登录 | P0 |
| 服务器历史 | 记录历史服务器地址，快速切换 | P1 |
| 连接检测 | 验证服务器地址有效性 | P1 |

**验收标准**：
- [ ] 输入正确的域名、账号、密码，能成功登录并跳转项目列表
- [ ] 登录失败时显示明确错误信息
- [ ] Token 过期后自动跳转登录页

#### 2.2.2 项目模块

**功能描述**：展示用户有权限访问的所有项目

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 项目列表 | 展示项目名称、描述、创建时间 | P0 |
| 项目搜索 | 支持按项目名称搜索 | P1 |
| 前端跳转 | 如项目配置了前端插件，显示跳转按钮 | P0 |
| 进入项目 | 点击进入该项目的 AI Chat 页面 | P0 |
| 下拉刷新 | 支持下拉刷新项目列表 | P1 |

**项目卡片展示信息**：
```
┌─────────────────────────────────────┐
│  项目名称                      [前端] │
│  项目描述内容...                     │
│  prefix: demo_project               │
│  创建时间: 2025-01-15               │
│                        [进入项目 →] │
└─────────────────────────────────────┘
```

#### 2.2.3 AI 对话模块

**功能描述**：通过自然语言与 AI 交互，查询和操作项目数据

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 发送消息 | 输入问题，发送给 AI 处理 | P0 |
| 消息展示 | 用户消息、AI 回复分左右显示 | P0 |
| 任务状态 | 显示 AI 任务执行状态和步骤 | P0 |
| 历史记录 | 查看历史对话记录 | P1 |
| 快捷指令 | 预设常用查询指令 | P2 |
| 任务重试 | 失败任务支持重试 | P1 |
| 取消任务 | 进行中的任务支持取消 | P1 |

**AI 能力示例**：
- "查看最新 10 条文章"
- "项目有多少个内容模型？"
- "今天有哪些定时任务执行了？"
- "查询用户表中 ID 为 123 的记录"

**任务状态流程**：
```
pending → running → completed/failed
                 ↘ cancelled
```

#### 2.2.4 内容浏览模块

**功能描述**：浏览项目中的内容模型和数据

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 模型列表 | 展示项目所有内容模型（列表型/单页型） | P1 |
| 内容列表 | 按模型查看内容数据列表 | P1 |
| 内容详情 | 查看单条内容的详细信息 | P1 |
| 单页内容 | 查看单页型模型的内容 | P1 |
| 分页加载 | 支持上拉加载更多 | P1 |
| 字段筛选 | 支持选择显示哪些字段 | P2 |

#### 2.2.5 系统监控模块

**功能描述**：查看云函数和定时任务状态

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 云函数列表 | 展示项目的 Web 端点函数 | P2 |
| 定时任务列表 | 展示定时任务及下次执行时间 | P2 |
| 状态筛选 | 按启用/禁用状态筛选 | P2 |

#### 2.2.6 个人中心

**功能描述**：账号管理和应用设置

| 功能点 | 说明 | 优先级 |
|--------|------|--------|
| 账号信息 | 展示当前登录用户信息 | P1 |
| 退出登录 | 清除本地 Token 并返回登录页 | P0 |
| 服务器管理 | 管理已保存的服务器地址 | P2 |
| 关于应用 | 版本信息、更新检查 | P2 |

## 3. 非功能需求

### 3.1 性能要求

| 指标 | 要求 |
|------|------|
| 首屏加载 | < 2s |
| 接口响应 | < 3s (正常网络) |
| 列表渲染 | 单页 20 条数据 < 100ms |

### 3.2 兼容性要求

| 平台 | 最低版本 |
|------|----------|
| iOS | 11.0+ |
| Android | 6.0+ |
| 微信小程序 | 基础库 2.20.0+ |
| H5 | Chrome 80+, Safari 13+ |

### 3.3 安全要求

- Token 本地加密存储
- 敏感操作需二次确认
- 支持 HTTPS
- 接口请求携带认证 Token

## 4. 用户流程

### 4.1 首次使用流程

```
启动应用 → 输入服务器地址 → 输入账号密码 → 登录成功 → 进入项目列表
```

### 4.2 日常使用流程

```
启动应用 → 自动登录 → 项目列表 → 选择项目 → AI 对话/内容浏览
```

### 4.3 AI 对话流程

```
进入项目 → Chat 页面 → 输入问题 → 发送 → 等待 AI 处理 → 查看结果
                                      ↓
                               显示任务状态 → 完成/失败/取消
```
