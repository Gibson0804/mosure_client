**交互说明**：

| 元素 | 交互 |
|------|------|
| 服务器地址输入 | 支持输入/粘贴，自动补全协议 |
| 历史服务器 | 点击展开下拉，选择快速填充 |
| 登录按钮 | 校验表单 → 请求登录 → 成功跳转/失败提示 |

- **加载中**：登录按钮显示 loading，禁用输入
- **登录失败**：Toast 提示错误信息
- **网络错误**：提示"无法连接服务器"

## 3. 页面跳转流程

### 3.1 主要流程图

```
                    ┌──────────┐
                    │  启动页   │
                    └────┬─────┘
                         │
                    检查登录状态
                         │
          ┌──────────────┴──────────────┐
          ↓                             ↓
    ┌──────────┐                 ┌──────────┐
    │  登录页   │←───退出登录────│ 项目列表  │
    └────┬─────┘                 └────┬─────┘
         │                            │
    登录成功                      选择项目
         │                            │
         └────────────→┌──────────────┘
                       ↓
                ┌──────────┐
                │ AI Chat  │
                └────┬─────┘
                     │
     ┌───────────────┼───────────────┐
     ↓               ↓               ↓
┌──────────┐  ┌──────────┐   ┌──────────┐
│ 对话历史  │  │ 内容模型  │   │ 系统监控  │
└──────────┘  └────┬─────┘   └──────────┘
                   ↓
              ┌──────────┐
              │ 内容列表  │
              └────┬─────┘
                   ↓
              ┌──────────┐
              │ 内容详情  │
              └──────────┘
```

### 3.2 页面参数传递

| 来源页 | 目标页 | 参数 |
|--------|--------|------|
| 项目列表 | AI Chat | `projectId`, `projectName`, `projectPrefix` |
| AI Chat | 对话历史 | `projectPrefix` |
| AI Chat | 内容模型 | `projectPrefix` |
| 内容模型 | 内容列表 | `projectPrefix`, `moldId`, `tableName`, `moldName` |
| 内容列表 | 内容详情 | `projectPrefix`, `tableName`, `contentId` |

## 5. 响应式适配

### 5.1 安全区域

```vue
<template>
  <view class="page">
    <!-- 状态栏占位 -->
    <view :style="{ height: statusBarHeight + 'px' }"></view>
    
    <!-- 页面内容 -->
    <view class="content">...</view>
    
    <!-- 底部安全区 -->
    <view :style="{ height: safeAreaBottom + 'px' }"></view>
  </view>
</template>

<script setup>
const { statusBarHeight, safeAreaInsets } = uni.getSystemInfoSync()
const safeAreaBottom = safeAreaInsets?.bottom || 0
</script>
```

### 5.2 键盘适配

```vue
<!-- Chat 页面键盘适配 -->
<template>
  <view class="chat-page" :style="{ paddingBottom: keyboardHeight + 'px' }">
    <scroll-view class="messages" :style="{ height: scrollHeight + 'px' }">
      ...
    </scroll-view>
    <view class="input-bar">
      ...
    </view>
  </view>
</template>

<script setup>
const keyboardHeight = ref(0)

uni.onKeyboardHeightChange(res => {
  keyboardHeight.value = res.height
})
</script>
```
